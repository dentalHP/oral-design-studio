Smart Start Pack 詳細説明

「ホームページを作りたいけれど、何から始めたらいいかわからない」「維持費を抑えて運用したい」という歯科医院様・個人事業主様向けの特別パッケージです。

このプランに含まれるもの

  • 現役DHによる監修： 専門用語を正しく、かつ患者様に伝わりやすい言葉で構成します。
  • 維持費¥0の仕組み： GitHub Pagesを活用し、月々のサーバー代がかからない構成をご提案します。
  • Googleマップ連携： MEO対策の第一歩であるマイビジネスへの登録をサポートします。
    • ※MEO対策とは、Googleマップでの表示順位を上げ、地域の患者様に見つけてもらいやすくする施策のことです。
  • スマホ完全対応： 全ての端末で美しく表示されるレスポンシブデザインです。
